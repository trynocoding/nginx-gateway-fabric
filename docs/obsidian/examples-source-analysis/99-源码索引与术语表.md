## 公共主链

| 阶段 | 源码入口 | 责任 |
|---|---|---|
| 事件入口 | `internal/controller/handler.go:eventHandlerImpl.HandleEventBatch` | 批处理 watch 事件 |
| 状态变更 | `internal/controller/state/change_processor.go:ChangeProcessorImpl.Process` | 更新集群状态并重建 Graph |
| 语义图 | `internal/controller/state/graph/graph.go:BuildGraph` | 引用解析、验证、状态条件 |
| 中间表示 | `internal/controller/state/dataplane/configuration.go:BuildConfiguration` | 转为协议无关的数据面模型 |
| 文件生成 | `internal/controller/nginx/config/generator.go:GeneratorImpl.Generate` | 生成 NGINX/证书/policy 文件 |
| 下发 | `internal/controller/nginx/agent/deployment.go:Deployment.SetFiles` | 推送给 agent 并驱动 reload |

## 术语表

| 术语 | 含义 |
|---|---|
| Graph | NGF 对 Kubernetes 期望状态完成引用解析和验证后的语义图，不等同于 API Server 对象缓存。 |
| dataplane.Configuration | Graph 到 NGINX 模板之间的中间表示。 |
| attachment | Route→listener 或 Policy→target 的受约束关联。 |
| ReferenceGrant | 目标 namespace 对跨 namespace 引用的授权。 |
| frontend TLS | 客户端到 Gateway listener 的 TLS。 |
| backend TLS | NGINX 到 Service backend 的 TLS 与身份验证。 |
| Provisioner | 根据 Gateway/NginxProxy 创建或更新数据面 Kubernetes 资源的链路。 |
