## 术语表

| 术语 | 含义 |
|---|---|
| NGF | NGINX Gateway Fabric，Gateway API 控制器 |
| Agent | NGINX Agent v3，运行在数据面 Pod 内 |
| Control Plane | 控制面，负责资源监听、配置生成、状态管理 |
| Data Plane | 数据面，负责真实流量处理 |
| MPI | Management Plane Interface，NGF 与 Agent 的 gRPC 协议 |
| CommandService | MPI 中负责连接、订阅、状态上报的服务 |
| FileService | MPI 中负责文件下载的服务 |
| CreateConnection | Agent 向 NGF 注册连接的 unary RPC |
| Subscribe | Agent 与 NGF 保持的双向 stream |
| ManagementPlaneRequest | NGF 发给 Agent 的管理面请求 |
| DataPlaneResponse | Agent 回给 NGF 的数据面响应 |
| ConfigApplyRequest | 控制面要求 Agent 应用配置的请求 |
| DeploymentStore | NGF 中保存数据面 Deployment 运行态的 store |
| Deployment | NGF 内部表示一个数据面 Deployment 的运行态对象 |
| Broadcaster | NGF 对订阅 Agent 广播配置更新的组件 |
| configVersion | 某次 NGINX 配置的版本标识 |
| instanceID | Agent/NGINX instance 的身份标识 |
| ResourceID | Agent 侧生成的数据面资源身份 |
| projected token | Kubernetes 投射给 Pod 的短期 service account token |
| token audience | token 预期访问对象，当前为 NGF Service DNS |
| GatewayClass | Gateway API 中声明控制器类别的资源 |
| Gateway | Gateway API 中声明入口监听器的数据面实例 |
| HTTPRoute | Gateway API 中声明 HTTP 路由规则的资源 |
