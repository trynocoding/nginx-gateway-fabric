# 19 函数值、闭包与高阶函数

## 语法

函数是一等值；闭包捕获词法环境，高阶函数把策略作为参数或结果传递。

**说明性片段：**

```go
double := func(v int) int { return v * 2 }

func apply(v int, f func(int) int) int { return f(v) }
```
