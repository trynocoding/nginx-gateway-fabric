# 24 接口的隐式实现

## 语法

类型只要方法集匹配就实现接口，无需 implements 声明；编译期断言能把意图显式化。

**说明性片段：**

```go
type Runner interface { Run() }

type Worker struct{}
func (Worker) Run() {}

var _ Runner = Worker{}
```

## NGF 中的应用

位置：`ngf:internal/framework/controller/reconciler.go:Reconciler interface assertion`

**原样源码：**

```go
var _ reconcile.Reconciler = &Reconciler{}
```

构造 *Reconciler → controller-runtime 按 reconcile.Reconciler 调用 → Reconcile 发送内部事件。
