# 04 有类型常量与枚举模式

## 语法

Go 没有 enum 关键字；常用“定义底层类型 + const 集合”获得类型检查和可读常量名。

**说明性片段：**

```go
type Level string

const (
	LevelInfo  Level = "info"
	LevelDebug Level = "debug"
)
```

## NGF 中的应用

位置：`ngf:apis/v1alpha1/nginxgateway_types.go:ControllerLogLevel`

**原样源码：**

```go
type ControllerLogLevel string
```

CRD JSON 字符串 → ControllerLogLevel → validateLogLevel switch → 日志级别更新。
