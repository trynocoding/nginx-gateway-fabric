# 12 切片共享底层数组与防御性复制

## 语法

赋值或截取切片不会复制元素；跨所有权边界时应 copy/Clone，防止别名修改。

**说明性片段：**

```go
dst := make([]string, len(src))
copy(dst, src)
```

## NGF 中的应用

位置：`ngf:internal/controller/nginx/config/servers.go:generateResponseHeaders`

**原样源码：**

```go
	responseRemoveHeaders := make([]string, len(headerFilter.Remove))

	// Make a deep copy to prevent the slice from being accidentally modified.
	copy(responseRemoveHeaders, headerFilter.Remove)
```

`responseRemoveHeaders` 先按源切片长度分配新数组，再用 `copy` 复制元素。返回值与 `headerFilter.Remove` 不再共享底层数组。
