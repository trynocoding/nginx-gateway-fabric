# 34 泛型和高阶函数组合

## 语法

泛型保存数据类型，高阶函数注入变化策略；组合后可复用控制流而不丢静态类型。

**说明性片段：**

```go
func Find[T any](items []T, match func(T) bool) (T, bool) {
	for _, item := range items {
		if match(item) { return item, true }
	}
	var zero T
	return zero, false
}
```
