# 03 变量声明与零值

## 语法

声明给类型分配零值：bool 为 false、指针/map/slice/channel 为 nil；构造函数只初始化不能依赖零值的成员。

**说明性片段：**

```go
var enabled bool      // false
var names []string    // nil
var count int         // 0
```

## NGF 中的应用

位置：`ngf:internal/controller/health.go:graphBuiltHealthChecker`

**原样源码：**

```go
ready   bool
```

newGraphBuiltHealthChecker 创建 readyCh → ready 保持 false 零值 → setAsReady 置 true 并关闭 channel。
