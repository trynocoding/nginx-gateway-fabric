# 33 泛型结构体与泛型函数类型

## 语法

泛型类型让容器保持元素静态类型；函数类型参数则让回调与结果类型一致。

**说明性片段：**

```go
type Page[T any] struct { Items []T }
type Predicate[T any] func(T) bool
```

## NGF 中的应用

位置：`ngf:internal/framework/waf/fetch/fetch.go:n1cPagedResult`

**原样源码：**

```go
type n1cPagedResult[T any] struct {
```

JSON 页面 → n1cPagedResult[具体响应项] → Items []T → 强类型谓词筛选。
