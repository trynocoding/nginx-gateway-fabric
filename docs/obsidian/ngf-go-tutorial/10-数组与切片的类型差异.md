# 10 数组与切片的类型差异

## 语法

数组长度属于类型且按值复制；切片是指向底层数组的描述符。

**说明性片段：**

```go
var digest [32]byte
bytes := digest[:] // []byte，仍引用 digest
```

## NGF 中的应用

位置：`ngf:internal/framework/helpers/helpers.go:URLHash`

**原样源码：**

```go
sum := sha256.Sum256([]byte(rawURL))
```

string 转 []byte → Sum256 返回 [32]byte 数组 → sum[:] 形成切片 → hex 编码。
