# NGF 源码中的 Go 1.26：系统教程与学习路线

> [!abstract]
> 这是一套给“能读懂 Go、但手写能力和知识体系有所退化”的工程师准备的源码教程。65 章各自只讲一个知识点，不设习题；每章都把语言机制落到当前 NGF 生产符号、调用效果和测试证据。

## 基线与读法

- 仓库：NGINX Gateway Fabric，revision `df175d68064b54c369b6cb2a6c83c8c1b2ab26ca`，取证时工作树干净。
- 语言：主模块与 tests 模块均为 Go 1.26.0；取证环境为 `go1.26.0 linux/amd64`。
- 语言事实以 [Go 1.26 Release Notes](https://go.dev/doc/go1.26) 和 [Go 1.26 语言规范](https://go.dev/ref/spec) 为准。
- 摘录均标注“原样摘录”；它们是上下文片段，不承诺脱离仓库独立运行。
- 主链：Kubernetes 事件 → Reconciler → EventLoop → ChangeProcessor → Graph → Dataplane → NGINX 配置。
