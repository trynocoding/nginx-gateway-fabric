# 31 方法约束

## 语法

接口约束可要求类型参数具备方法集；泛型代码因此能静态调用这些方法。

**说明性片段：**

```go
type Named interface { Name() string }

func PrintName[T Named](v T) { fmt.Println(v.Name()) }
```

## NGF 中的应用

位置：`ngf:internal/controller/state/store.go:objectStoreMapAdapter`

**原样源码：**

```go
type objectStoreMapAdapter[T client.Object] struct {
```

具体 Kubernetes 对象指针满足 client.Object → 适配器可调用对象方法 → 统一进入 objectStore。
