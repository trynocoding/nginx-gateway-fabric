# 32 类型集合、~ 与联合约束

## 语法

联合 A|B 枚举允许类型；~T 还接受底层类型为 T 的自定义类型。

**说明性片段：**

```go
type Text interface { ~string }
type Number interface { ~int | ~int32 }
```

## NGF 中的应用

位置：`ngf:internal/controller/state/dataplane/convert.go:convertMatchType`

**原样源码：**

```go
func convertMatchType[T ~string](matchType *T) MatchType {
```

不同 API 包的字符串新类型 → T ~string 统一接收 → 转换为 dataplane MatchType。
