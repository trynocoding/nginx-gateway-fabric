## 术语表

| 术语 | 本教程中的固定含义 |
|---|---|
| API 模型 | Kubernetes/CRD 对外序列化类型，主要位于 `apis/`。 |
| ClusterState | ChangeProcessor 持有的当前 Kubernetes 对象视图。 |
| Graph | 经过引用解析、校验和附着关系计算的领域模型，不是 nginx.conf。 |
| Dataplane Configuration | 面向 NGINX 生成器的中间表示。 |
| Reconciler | controller-runtime 回调；NGF 中主要把资源状态翻译成内部事件。 |
| EventLoop | 拥有 current/next 双缓冲批次的事件循环。 |
| ChangeProcessor | 捕获资源变更并按需重建 Graph 的状态组件。 |
| Predicate | 入队前过滤 Kubernetes 事件的函数或对象。 |
| FieldIndexer | controller-runtime Cache 中的客户端字段索引。 |
| 所有权 | 谁可修改、关闭、取消或释放某个值或资源。 |
| 源码事实 | 在固定 revision 的生产源码中直接可见。 |
| 测试佐证 | 被命名测试覆盖或断言的行为。 |
| 当前仓库未采用 | Go 1.26 支持、但在生产源码普查范围内没有对应用法。 |
