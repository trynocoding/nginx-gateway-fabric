# 36 %w 与错误链

## 语法

fmt.Errorf 使用 %w 在增加语境的同时保留可遍历的原因链。

**说明性片段：**

```go
if err != nil {
	return fmt.Errorf("load config: %w", err)
}
```

## NGF 中的应用

位置：`ngf:internal/framework/file/file.go:Write`

**原样源码：**

```go
return fmt.Errorf("failed to create file %q: %w", file.Path, err)
```

底层文件错误 → 加入路径与动作 → 上层返回或聚合 → 日志获得可读上下文。
