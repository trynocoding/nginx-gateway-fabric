# 25 小接口与依赖注入

## 语法

接口应由消费者按最小能力定义；构造配置注入实现，测试便可替换 fake。

**说明性片段：**

```go
type Reader interface {
	Get(context.Context, string) ([]byte, error)
}

type Service struct { reader Reader }
```

## NGF 中的应用

位置：`ngf:internal/framework/kubernetes/client.go:Reader`

**原样源码：**

```go
type Reader interface {
```

业务组件依赖 Reader → 生产注入 controller-runtime client → 测试注入 FakeReader。
