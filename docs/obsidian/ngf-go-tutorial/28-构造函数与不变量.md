# 28 构造函数与不变量建立

## 语法

Go 构造函数只是普通函数；价值在于初始化 map/channel/context 等不允许为零的状态。

**说明性片段：**

```go
func NewCache() *Cache {
	return &Cache{items: make(map[string]string)}
}
```

## NGF 中的应用

位置：`ngf:internal/controller/nginx/agent/broadcast/broadcast.go:NewDeploymentBroadcaster`

**原样源码：**

```go
listeners:         make(map[string]storedChannels),
```

调用构造函数 → 初始化 channels/map/child context → 启动 subscriber 和 publisher → 返回可用对象。
