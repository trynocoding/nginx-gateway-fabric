# 29 泛型函数与类型推断

## 语法

类型参数让同一算法在静态类型下复用；参数通常可从实参推断，无需写方括号。

**说明性片段：**

```go
func Pointer[T any](v T) *T { return &v }

p := Pointer(42) // 推断 T 为 int
```

func GetPointer[T any](v T) *T {
