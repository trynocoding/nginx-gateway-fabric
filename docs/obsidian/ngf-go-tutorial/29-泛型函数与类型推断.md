# 29 泛型函数与类型推断

> [!abstract] 本章唯一知识点
> 类型参数让同一算法在静态类型下复用；参数通常可从实参推断，无需写方括号。

## 前置与完成标准

前置：[[28-构造函数与不变量]]。学完应能解释“泛型函数与类型推断”，并在 NGF 中定位 `GetPointer`，说清调用效果与适用边界。本章只做源码导读，片段不承诺可独立运行。

## 工程取舍与边界

单一值转指针算法不应为每种 API 类型复制。

> [!warning] 常见误解与迁移边界
> 可复用形状相同的算法；领域语义不同不要只因签名相似而泛化。误解是泛型等于运行期反射。

复用判定：明确可复用的做法可直接采用；带条件的做法必须先满足相同输入、所有权与失败语义；指出不要或不可的部分不应照搬。
