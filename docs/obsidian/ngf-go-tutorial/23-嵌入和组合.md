# 23 嵌入和组合

## 语法

匿名嵌入会提升字段/方法，但仍是组合，不是继承；外层可覆盖被提升的方法。

**说明性片段：**

```go
type Base struct{}
func (Base) Enabled() bool { return true }

type Service struct{ Base }
```

## NGF 中的应用

位置：`ngf:internal/controller/nginx/config/policies/waf/generator.go:waf.Generator`

**原样源码：**

```go
policies.UnimplementedGenerator
```

WAF Generator 嵌入默认实现 → 只实现需要的生成方法 → 满足 policies.Generator。
