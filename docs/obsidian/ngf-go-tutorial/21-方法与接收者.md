# 21 方法与接收者

## 语法

方法把函数绑定到已定义类型；接收者决定它操作哪个对象状态。

**说明性片段：**

```go
type Counter int

func (c Counter) Value() int { return int(c) }
```

## NGF 中的应用

位置：`ngf:internal/controller/health.go:graphBuiltHealthChecker.readyCheck`

**原样源码：**

```go
func (h *graphBuiltHealthChecker) readyCheck(_ *http.Request) error {
```

controller-runtime 调 readyCheck → 读 h.ready → 返回 readiness 结果。
