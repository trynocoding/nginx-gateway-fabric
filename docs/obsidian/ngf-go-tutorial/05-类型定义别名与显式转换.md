# 05 类型定义、类型别名与显式转换

## 语法

类型定义创建新类型；类型别名只是原类型的另一个名字。两个已定义类型即使底层类型相同，互相赋值时通常也要显式转换。

**说明性片段：**

```go
type UserID string
type Alias = string

id := UserID("42") // 显式转换
```

## NGF 中的应用

位置：`ngf:apis/v1alpha2/nginxproxy_types.go:ServiceType`

**原样源码：**

```go
type ServiceType corev1.ServiceType
```

Kubernetes ServiceType → 显式转换为 NGF ServiceType → CRD 校验与生成逻辑消费。
